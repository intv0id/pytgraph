Developer install¶

To install a developer version of pytgraph, you will first need to clone the repository:

git clone https://github.com//pytgraph
cd pytgraph

Next, install it with a develop install using pip:

pip install -e .

If you are planning on working on the JS/frontend code, you should also do a link installation of the extension:

jupyter nbextension install [--sys-prefix / --user / --system] --symlink --py pytgraph

jupyter nbextension enable [--sys-prefix / --user / --system] --py pytgraph

with the appropriate flag. Or, if you are using Jupyterlab:

jupyter labextension install .
